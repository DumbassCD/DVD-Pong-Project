# DVD-Pong-Project-
HTML ping pong with theme of DVD/CD media with some references to Sonic and other games with many gameplay things that are developed by best buds and leaders of Sonic Classic Team studio.
Theres 2 different branches over there, because the project are still developed. Main is a branch for project files, other Level 2-3 and other are for level files.
The project started in our IT School's class and tools used are MS Paint (Due its simplicity and that there was no Paint.net on school laptop i used it, and its fun) for graphics + actual Paint.net for other graphics, Notepad++ for coding!

Little notes:
1) The .html files in the root folder is purely for source, if u want to test, use the Main Build folder.
